Website

  • A website is collection of related webpages.
  • It contains text, images, graphics, videos (multimedia elements) etc.
  • The website loads with its default webpage called home page of the website.
  • The related webpages are linked together using concept of hypertext and hyperlinks.

Working of Website

Step-1 :- Firstly user makes request for accessing a website by typing its domain name (web address or URL) in the address bar of the browser.

Step-2 :- The browser sends request to DNS (Domain Name System) server for looking IP address of the domain name.

Step-3 :- The DNS translates human readable domain name into numeric IP address (For Example – 168.11.129.64 ).

Step-4 :- The DNS server returns the ID address to the browser.

Step-5 :- The browser now sends the request for webpage using IP address to connect to the web server.

Step-6 :- The web server responds the request made by web client (browser) and returns the copy of the webpage.

Step-7 :- The web page is now display on the browser.

Types of Website

  1. Static Website
  2. Dynamic Website

Static Website

  • Static Website are those website which have predefined and fixed contents and remain save every time when user success the website.
  • Usually these websites are informative website and do not require any information or data to be stored in the data base.
  • The contents of static website can be changed manually at server site. The user can not modify it.
  • Static Website are design using only front end tools (HTML, CSS, JAVA SCRIPT).

Dynamic Website

  • Dynamic Website are those website whose contents changes dynamically on the basis of user request (in real time manner).
  • Dynamic website are create using front end and back end tools.
  • Example of Dynamic Websites are – Online exam form/registration form, social sites like facebook, twitter etc. E-commerce site like flipkart, amazon etc. IRCTC or online booking system, online banking system site.

Front End

  • It refers to client side design of website. Front end is involved with what the user see and interact with website i.e. user interface (UI) which includes the design, layout, navigation menu, image, text, videos, image, gallery etc.
  • Technology involved in front end design and development are HTML, CSS, JAVA SCRIPT, W3CSS (frame work of CSS, Bootstrap (frame work of java script), React JS (frame work of Java Script) , J Query (library of JS).

Back End

  • Back End refers to design and development of website at server side.
  • It is involved with what user cannot see in the browser like database, server information etc.
  • It concerns with security aspects, data storage and manipulation, content management, user authentication etc.
  • Technology involved in back end development are Python, PHP (hyper text Processor), ASP (Active server Pages), Ruby etc.
  • Some popular framework for back end are Django Python, express JS, Nude JS etc.

Scripting Language

Script or scripting language is a programming language with a series of commands which is interpreted and automates the execution of tasks.
It is also following two types-

  1. Client Side Scripting
  2. Server Side Scripting

Client Side Scripting

Client Side Scripting Language are those language that are executed on client side without server side processing the browser to run the scripts.
For e.g.- It can be used to validate user data before submitting to the server. It minimizes the loads of the server. For e.g.- HTML, CSS, Java script.

Server Side Scripting

These language are executed on web server of scripts. It deals with database and provides dynamic content to the web-page these on the user request.
For e.g.- PHP, Python, Java, ASP

Note :-

  • The client side scripting language deals with front end of the website where as server side scripting language deals with back end of the website.
  • The client side scripting languages are used for designing and developing dynamic websites.

Responsive Web Designing

Responsive Web Design means that designing of website in such a way that contents of the websites automatically fit and resize that makes the website looks good on all devices having different screen sizes such as (PC/Computer/Laptop/Tablets/Mobile Phones) wherever it is opened.
HTML, CSS and other frameworks are used to make website responsive.
