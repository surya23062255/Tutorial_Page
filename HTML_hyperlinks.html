Hyperlinks in HTML

  • Hyperlinks are used to jump into another page or an specific location of some webpage by clicking on it.
  • <a> tag ("a" stands for anchor) is use to define hyperlink and hypertext in HTML. A link can be text, images, video or any other document.
  • A Hyperlink is a link reference that points to another webpage or another location of same webpage. A Hypertext is that text on webpage that contains hyperlink.
  • By default hyperlinks are blue color and underline. When the mouse move over hypertext, the mouse arrow turns into little handshape and by clicking on it user can move to another location.

Three states of links (by default)

  • Unvisited Link → Blue and Underline
  • Active Link → Red and Underline
  • Visited Link → Purplr and Underline

Attributes of <a> tag

  • href :- It is the hypertext reference that defines the address or path of the file which id hyperlinked.
  • target :- It defines where to open the linked document. It can have following values-
    1. _blank :- It opens the linked document in a new tab or window.
    2. _self :- It opens the linked document in same tab or window. It is default value.
    3. _top :- It opens the linked document in the full body of windows.
    4. _parent :- It opens the linked document in parent frame.
  • title :- This attribute specifies a short title or extra information when the mouse over the hypertext.

Linking within same page

HTML bookmarks are use to jump over a specific part of web page. They are useful when webpage is very long and have lots of contents.

For linking within same page follow these steps-

  • STEP-1 : Create the bookmark by using id attribute for the element.
         For e.g. <h1 id="ch1">chapter1</h1>
  • STEP-2 : Create hyperlink to jump over the bookmark. by using hash (#) symbol with the value of      href attribute (# refers to the id).
         For e.g. <a href="#ch1">Go to chapter 1</a>
