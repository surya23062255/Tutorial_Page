Hyper Text Markup Language (HTML)

  • HTML stands for Hyper Text Markup Language.
  • It is language for creating webpages and websites.
  • It specifies the structure of webpage.
  • The basic building block of HTML are tags or elements and their attributes.
  • Web browsers use them (tags) to render the contents of webpage.
  • HTML 5 is the latest version of HTML. It is maintained by W3C (World Wide Web Consortium).
  • HTML 5 has new and latest feature that makes it more powerful and dynamic while designing the websites. New tag and their attributes are included in HTML 5, some elements are removed some are redefined from previous version (HTML4).
  • In 1991 Tim Berners Lee invented HTML.

Note :-

  • HTML is not a case sensitive language.
  • Extension of HTML file/document or web page/document is .html or .htm

HTML tags or elements

Tags are markup and coded instructions written as source code. It is enclosed in angular brakets (<> ). Usually there are opening (start) and closing (end) tags with containing some text or object in between.

Types of tags

There are two types of tags

  1. Container Tags
  2. Empty Tags

Container Tags

Container tags are those tags that have both opening and closing tags. These tags have some text or object between opening and closing tags. Opening tag ON the effect and closing tag OFF the effect.

Syntax :- <tagname> Some text or object <tagname>
Example :-
<h1> This is heading <h1>
<p> This is paragraph <p>

Empty Tags

Empty tags are those tags that have only opening tag and don’t have closing tag. They do not contain any text or object because it don’t have closing tag.

Example :-
<br> - For line break
<hr> - For Horizontal rular

Text Formatting Elements

Tags/Elements Description
<h1> to <h6> Levels of headings, where <h1> is the largest and <h6> is the smallest level of heading.
<p> Defines Paragraph
<b> To bold/dark the Text
<i> Itallic Text
<u> Underline Text
<strong> Strong text (same as <b>) with some extra importance
<small> Smaller text than other text (in size)
<big> Bigger text than other text (in size)
<em> Emphasized Text
<mark> Marked Text (Usually with yellow background behind the text)
<ins> Inserted Text
<del> Deleted Text
<strike>or <s> Strikethrough Text (similar to <del>)
<sub> Subscripted Text (For Example : H2O)
<sup> Superscripted Text (For Example : a2+b2)
<tt> Teletype (Mono Space) text, which means that each letter is displayed with equalfixed size width.
Note :- All the above text formatting tags are container tags.
<pre> It is use to define pre-formatted text. It prserves the spaces and line breakes as in the code
<br> For line break
