Internet

  • The internet is a globally connected network system that uses TCP/IP (Transmission Control Protocol/Internet Protocol) model to transmit data via various type of media (Wired or Wireless).
  • It is a network of networks (Inter Networking).
  • It is combination of private, public, business, academic, government etc. the networks connected together millions of millions people are uses of Internet.
  • It has complex combination of several technology and services use around the world every day.

History of Internet

  • In 1967 in department of defence (DOD) of US Government started a project called ARPANET (Advance Research Projects Agency Network) that was mainly related to small network of computers.
  • The idea was that each host computer connected to specialized computer called IMP (Interface Model Processor/Interface Message Processor) to create a network.
  • In 1969 ARPNET connected four notes from different locations via IMB to from a network.
  • In 1970 a communication model (TCP/IP) that set standard for transmission of data between multiple networks.
  • ARPANET adopted TCP/IP model on 1st January 1983 globally and network of networks came into signification that become model internet gradually.

World Wide Web (WWW)

  • The World Wide Web (WWW) or web is the service of internet that are used for accessing data online in the form of website and hyperlinks.
  • It is repository of information in which the documents are distributed all over the world and related are linked together.
  • It works on distributed clients server service in which the clients using browser can access a service from server.
  • The WWW was developed by Tim Berners Lee in 1989 and commercial started in 1990.

Web Client

  • Web browser is known as web client that interpret and display the web pages.
  • It sends request to the server for particular resource.
  • Example : Some browser are Internet Explorer, Edge, Google Chrome, Mozilla Firefox etc.

Web Server

  • Web server is a computer where web pages are stored. Web server contains software that responds to the request for resources made by web clients.
  • It is used to host website. The Hypertext Transfer Protocol (http) defines the rules for how to communication takes place between web client and web server.

Difference between Internet and WWW

Internet WWW
Internet originated sometimes in late 1960s. Time Berners-lee invented the world wide web in 1989.
Nature of internet is hardware. Nature of WWW is software.
Internet consists of computers, routers, cables, servers, cellular towers etc. WWW consists of information like text, images, audio, video.
The first version of the internet was known as ARPANET. In the begining WWW was known as NSFNET.
Internet works on the basis of Internet Protocol (IP). WWW works on the basis of Hypertext Transfer Protocol (HTTP).
Internet is Superset of WWW. WWW is a Subset of internet.

Web Page

Web Page is a single document or page available on WWW. The web page are created using HTML and are interpreted and displayed by web browsers. Each web page has unique address called URL.

Uniform Resource Locator (URL)

  • A URL is the address of a specific webpage or file on the internet.
  • It separate one page to another.
  • It is a unique identifier for a web page or file on www.
  • The URL has following parts –
    • Protocol
    • Host
    • Port
    • Path

Syntax :-

  • Protocol://host/path
  • Protocol://host:port/path
Example :-  http://www.google.com
