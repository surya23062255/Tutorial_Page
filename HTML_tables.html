HTML tables

Tables are the combination of rows and columns. Rows are arranged horizontally and columns are arranged vertically. The intersection point of a row and column is called cell where we actually inter the data.

The <table> tag is use to define a table. Each row of the table is defined by <tr> tag and each row of the table is defined by <td> tag. The header cell is defined by <th> tag.

Attributes of <table> tag

  • Border :- It specifies border around the table.
  • Height :- It specifies the height of the table.
  • Width :- It specifies the height of the table.
  • Cellspacing :- It specifies space between table border and cell border.
  • Cellpadding :- It specifies space between cell border and contents of the cell.
  • Align :- It adjust alignment of the table. It takes value either Left(default), Right and Center.


Caption in the Table

The <caption> tag is used to give a title/name/short information of the table. It provides short description to the table. It takes two value with align attributes: Top (default) and bottom.

Syntax : <caption>............</caption>

Colspan and Rowspan Attributes (only use with <td> or <th> tag)

  • Rowspan : This attribute spans the cell into rows.
          For e.g. <td rowspan="3">........</td>
          This spans the cell into two rows.
  • Colspan : This attribute spans the cell into columns.
         For e.g. <td colspan="2">........</td>
         This spans the cell into two columns.
