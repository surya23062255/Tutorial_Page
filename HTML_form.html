HTML Form

HTML form or web form or online form is use to collect users input data. For example Login Form, Registration Form etc.
Users enter or fill their information using form element such as text boxes, radio buttons, check boxes, drop down menus etc.
The <form> tag defines a form in HTML.

Syntax :

The form element can contain one or more of the following form elements or controls.

Form Elements

  • <label>
  • <input>
  • <button>
  • <textarea>
  • <select> and <option>
  • <fieldset> and <legend>
  • <optgroup>

<label> tag

The <label> tag is used to specify a label for an <input> element of a form. It adds a label to a form control such as text, email, password, textarea etc. It toggles the control when a user clicks on a text within the <label> element.

<input> tag

The <input> tag or element is use to create many different fields on the basis of type attribute.

<input> Element Types Description Output
<input type= "text"> Defines a text box
<input type= "password"> Defines a textbox as password field. It masks the characters
<input type= "radio"> Defines a radio button Male
<input type= "checkbox"> Defines a checkbox Terms & conditions
<input type= "number"> Defines a number field
<input type= "file"> Defines a file upload or browse button
<input type= "email"> Defines a E-mail field
<input type= "date"> Defines a date picker
<input type= "color"> Defines a color picker Choose any color
<input type= "button"> Defines a normal/event button
<input type= "reset"> Defines a reset button (clear all form fields at once)
<input type= "submit"> Defines a submit button
<input type= "image"> Defines a image button as submit button

Attributes of <input> tag

  • Value : It specifies the initial value oif element.
  • Name : It specifies name of the element.
  • Placeholder : It specifies the hint for input that is given by the user.
  • Readonly : It specifies the input field is read only. It means we cannot enter any data or value.
  • Disabled : It specifies that the input field is disabled or inactive. It also does not allow to enter any data or value.
  • Autocomplete : It can have value "ON" or "OFF" when user starts typing in the input field, it predicts the texts data typed earlier.
  • Max : It specifies the maximum value in the input field.
  • Min : It specifies the minimum value in the input field.
  • Maxlength : It specifies the maximum length (No. of characters) of input field.
  • Size : It controls the width of the input field.
  • Checked : This attribute checks the option automatically when form loads (it is used with radio buttons and checkboxes).
  • Required : It specifies the input field must be filled before the submitting the form.
  • Selected : It specifies pre selected option when page loads (used while creating drop down menu).

Pattern Attribute

This attribute specifies a regular expression that the input fields value check against on form submission.
We can use title attribute to describe the pattern that help to user.

Attribute Values Description
[a-zA-Z] Allow alphabets (upper & lower case). Any number of characters.
[a-zA-Z]{3} Allow alphabets upper &lower case with exactly 3 characters.
[a-zA-Z]{3,} Allow upper & lower case alphabets with minimum 3 characters and maximum no limit.
[a-zA-Z0-9]{3,8} Allow upper & lower case alphabets and numeric digits with minimum3 and maximum 8 characters.
[0-9]{10} Allow numeric digits only with exactly 10 digits.
.{5,15} Allow typer of characters with the range of minimum 5 and maximum 15 characters.
