HTML Lists

Lists are use to group a set of related data items. It is of three types.

  1. Unorderd List
  2. Ordered List
  3. Definition List

Unorderd List

It is collection of related data items that have no special order or sequence.
The list is created by using <ul> tag and each list item starts with <li> tag. The list items is marked with bullets.

The Type attribute of <ul> tag is used to specify the type of bullet that we want. It takes three values.

  1. <ul type= “disk”> (by default)
  2. <ul type= “circle”>
  3. <ul type= “square”>

Ordered List

This type of list is a collection of related data items that have order or sequence. This list is created by using <ol> tag, and each list item starts with <li> tag.

The Type attribute of <ol> tag is used to specify the type of numbering that we want. It takes the following possible values.

  1. <ul type= “number”>  →  Number (by default)
  2. <ul type= “A”>     →  Upper case letters
  3. <ul type= “a”>     →  Lower case letters
  4. <ul type= “I”>     →  Upper case Roman Numbers
  5. <ul type= “i”>    →  Lower case Roman Numbers
We can use start attribute with <ol> tag to specify the starting point of numbering we need.
